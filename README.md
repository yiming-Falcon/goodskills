# goodskills

A personal GitHub repository for sharing mirrored and collected agent skills.

## Skills

### 1. web-access
- Share link: https://github.com/yiming-Falcon/goodskills/tree/main/skills/web-access
- Install:

```bash
npx skills add yiming-Falcon/goodskills/skills/web-access
```

- Agent install:

```text
帮我安装这个 skill：https://github.com/yiming-Falcon/goodskills/tree/main/skills/web-access
```

## Why people often provide both an install line and a URL

The install line is for a CLI or tool to execute directly.
The GitHub URL is for people or agents to inspect the source, understand what will be installed, and use as a fallback when direct install is not supported.
